Custom Library

The Song Shaker App allows you to add songs to your custom song library via the Add Song page.

Your current custom library is displayed below.

You may update your settings using the checkboxes to the left of each song.

Active Songs

If you do not want an active song to be available for use in your set lists, please click its corresponding "deactivate" checkbox to the left of the song details.

Inactive Songs

If you would like to make an inactive song available for use in your set lists, please click its corresponding "activate" checkbox to the left of the song details.
