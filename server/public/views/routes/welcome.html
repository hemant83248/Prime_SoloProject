Welcome to Song Shaker!

Song Shaker makes it easy to generate a set list for your next gig!

Song Shaker comes with a standard library of over 200 songs. We give you the ability to choose which of those songs you would like to be available when creating your sets.

You can also add as many custom songs as you would like to make Set Shaker work best for you!

Take a tour around the app:

  • Visit the Set List Generator to check out the awesome set lists you can make.
  • Visit the Standard Song Library and View Custom Library Page pages to see what songs are available to you.
  • Visit the Add Song page to start adding songs to your Custom Library.
